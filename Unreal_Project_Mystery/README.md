## 制作物について
### 概要 
Unreal Engineで動作する、3D謎解きアクションゲームです。  
白い球体に触れると、クリアするために必要な情報を得ることが出来ます。
道中にちりばめられた謎を解き明かし、クリアを目指しましょう。

### 基本操作
- **移動**: WASDキー  
- **ジャンプ**: スペースキー  
- **アクション**: 左クリック  
　※上手く動かないときは、左クリックをしてみてください、

## 使用フレームワーク
Unreal Engine4.27
